5. OpenCV  많이 사용하는 명령어
   (1) waitKey(기다리는 시간)
       입력값이 있을때 까지
       0 : 무제한 기다림.
       
   (2) 이미지를 회색으로 처리하는 것
       함수 : cv2.cvtColor
       옵션 : COLOR_BGR2GRAY
   (3) 컬러나 색상을 바꿀때 옵션
       cv2.COLOR_
       <- 색상변경과 관련된 옵션

   빛의 3요소 : RGB
   OpenCV : BGR2GRAY

   (4) imshow
       이미지를 화면에 표시
       imshow('윈도우창이름', '이미지')

   (5) flip(이미지, 어떻게 반전?)
       이미지 반전
       옵션
            < 0 : 상하좌우 대칭
            = 0 : 상하 반전
            > 0 : 좌우 반전
